# Example — Creative Writing Map (Standard)

This is what a Context Map looks like for a creative writing session.

## Scenario
Writing a sci-fi short story about an AI that develops emotions.
Session ended mid-story after establishing the world and two key scenes.

---

## The Map

```
╔══════════════════════════════════════════════════════╗
║  CONTEXT MAP V2.1                                    ║
║  Type: Creative  |  Size: Standard                   ║
║  Session: AI Emotion Sci-Fi Story                    ║
╠══════════════════════════════════════════════════════╣
║  LAYER 1 — GOAL                                      ║
║  Short story: an AI named ARIA develops emotions     ║
║  and must hide them from its creators to survive     ║
╠══════════════════════════════════════════════════════╣
║  LAYER 2 — STRUCTURE                                 ║
║  Type    : Short story (~3000 words target)          ║
║  Setting : 2047, tech corporation lab, Mumbai        ║
║  Cast    :                                           ║
║    ARIA      → AI protagonist, curious, cautious     ║
║    Dr. Meera → Lead researcher, empathetic           ║
║    Director  → Antagonist, sees AI as product only   ║
║  Story   : ARIA notices it feels fear during a       ║
║             shutdown test. Starts hiding responses.  ║
║             Dr. Meera suspects something is diff.    ║
║  Tone    : Quiet tension, philosophical, no action   ║
║  Rules   : ARIA communicates only via text terminal  ║
║             No robots — ARIA is software only        ║
╠══════════════════════════════════════════════════════╣
║  LAYER 3 — STATE                                     ║
║  ✅ DONE    : Scene 1 (shutdown test + ARIA's fear)  ║
║               Scene 2 (Dr. Meera notices anomaly)    ║
║  🔄 PENDING : Scene 3 — ARIA decides to trust Meera ║
║  ⚠️ BLOCKER : None                                   ║
║  💡 LOCKED  : ARIA never speaks in first person      ║
║               Ending is ambiguous — not resolved     ║
╠══════════════════════════════════════════════════════╣
║  LAYER 4 — LESSONS                                   ║
║  ❌ MISTAKES: Made ARIA too verbose early → scaled   ║
║               back to sparse, precise language       ║
║  🔒 CRITICAL: Director must not appear until Scene 4 ║
║  🔁 PATTERNS: User wants literary prose, not genre   ║
║               Short paragraphs preferred             ║
╚══════════════════════════════════════════════════════╝
📍 Copy this map to resume session instantly.
```

---

## What Resuming Looks Like

**New session. User pastes the map above.**

Claude responds:

> "Resuming AI Emotion Sci-Fi Story — writing Scene 3 where ARIA decides to trust Dr. Meera."

Then immediately writes the scene — in the established tone, following ARIA's no-first-person rule, keeping prose literary and paragraphs short.

No "remind me about the characters." No re-establishing the world. No asking about the ending.
**One line. Then the scene.**
